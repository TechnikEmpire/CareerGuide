# CLAUDE.md

Follow the canonical repository instructions in `AGENTS.md`.

Следуйте каноническим инструкциям репозитория в `AGENTS.md`.

Key rule summary:

- This project is a web-based academic career-guidance proof of concept.
- Enduring documentation must be bilingual in English and Russian.
- Backend work should default to Python.
- Frontend work should default to JavaScript or TypeScript.
- Keep code modular, readable, and well-commented where the logic is non-obvious.
- Record architectural changes in `docs/DECISIONS.md`.

Краткое резюме правил:

- Это академический proof of concept веб-приложения для карьерного сопровождения.
- Долговечная документация должна вестись на английском и русском языках.
- Backend по умолчанию следует писать на Python.
- Frontend по умолчанию следует писать на JavaScript или TypeScript.
- Код должен быть модульным, читаемым и хорошо прокомментированным там, где логика неочевидна.
- Архитектурные изменения нужно фиксировать в `docs/DECISIONS.md`.
